# Lecture Material

The following material is covered in the lectures:
* [L01](./L01/README.md): C fundamentals for experienced programmers.
* [L02](./L02/README.md): Hardware-near programming in C: registers and bitwise I/O.
* [L03](./L03/README.md): Interrupts: external interrupts and pin-change interrupts.
* [L04](./L04/README.md): Analog-to-digital conversion (ADC).
* [L05](./L05/README.md): Timers, non-blocking delays and periodic events.
* [L06](./L06/README.md): Serial communication (UART).
* [L07](./L07/README.md): EEPROM.
* [L08](./L08/README.md): The watchdog Timer.
* [L09](./L09/README.md): Structs as drivers.
* [L10](./L10/README.md): Encapsulation and polymorphism in C.
